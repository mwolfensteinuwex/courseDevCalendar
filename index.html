Start Planning Faculty Retreat

Propose a mandatory kick-off meeting with Program Managers, ID, & Media. This is usually the Program Manager's responsibility.

Program managers have leveraged Doodle in the past to pick dates for faculty retreats.

Generate Media Reports for Course Revisions

A media report will be generated for courses up for revision the following fiscal year. A media report is a comprehensive evaluation of media components currently being leveraged in a specific course. The types of information you can expect to find in a media report include, but are not limited to, current media player versions, audio quality, appearance, readability, transcripts, and captions.

This report provides media and instructional design with a clear plan for upgrading media components (where applicable) if media components are reused in a revised course.

Assets: Working example and Media Report - Template

Faculty Retreat

Faculty get face time with other faculty, instructional design, media, the program manager, student services, and possibly other team members at this program-manager led meeting.

This is a good time for instructional designers and media to have informal, face-to-face meetings (especially with new faculty).

Course Structure Finalized

The course outline or course structure should be finalized by this date. At the very least, this includes a list of lessons or topics to be covered.

If not completed, this is a potential flag for the Program Manager.

Resource: Example of Course Structure

ID Briefs Media on Courses

ID notifies media of the number and type of media for each new or revised course, and consults with media about any unknowns and schedules media plan consultation with faculty if needed (see Establish Media Plan).

Initial Course Design Complete and Development Begins

This is the ID and instructor's plan for the course. This is a pre-requisite to begin development of the course. The ID has completed a consultation (or needs assessment) and has worked together with the instructor to identify the different components of the course and pedagogical strategies to drive development.

For an example of components commonly found in a course: Instructional Design Wiki (Development)

Textbook(s) Selected (if any)

If the instructor is using a required textbook or course reader, that information should be submitted to the ID by this time. ID should notify student services so they can send updates to Neebo, the bookstore.

Course Live

100% of the course is finished by the day the course opens (Preview Week).

Consultation to Establish Media Plan

ID schedules a phone conference with the instructor and the lead media specialist. The objective of the meeting is to share ideas about media usage and finalize production details and schedules.

The media specialist should have enough information to accurately fill out the media plan for the course following the meeting.

The Media team is planning to develop a document to more effectively capture information for the media plan.

Resource: If an instructor is creating Storybook presentations, send them the Storybook Presentation Template

Resource: If an instructor wants to schedule a video shoot, sent them the CEOEL Course Video Form

Media Consultation

The lead media specialist schedules an Adobe Connect web conference with the instructor to discuss media development procedures. Examples may include, but are not limited to, recording audio in PowerPoint or other presentation applications, configuring screencast software for lecture recordings, and/or planning video shoots.

One of the main objectives of this web conference is to encourage the instructor to create a media test file and submit it via our media uploader web application. Upon submission, a media specialist will review the test file and provide feedback to the instructor.

Resource: Adobe Connect Account

First Media Sample/Test Submitted

Following the Media Consultation, the instructor will develop and submit a sample media file for review. It is the responsibility of the lead media specialist to perform a quality assurance check on the submission to ensure it meets our standards.

NOTE: ID will need to inform media when the file is ready for review since they receive alert notifications from the media uploader web application.

Resource: Media Upload Site Password is Password will be added here

Media Sample/Test Response

The media specialist will follow up with the instructor regarding the media test file they submitted. If the media quality (audio, visuals, output, etc.) meets our standards, the instructor will be granted the go-ahead to begin media production for their respective course. If problems exist, the media specialist will schedule a follow-up media consultation to troubleshoot issues. This process will continue until a high-quality product is delivered.

NOTE: Performing multiple media consultations is a rare situation, and to date we have never performed more than three for any one instructor.

25% of Content Submitted for Development

This date signifies that roughly one fourth of the course content has been submitted by the instructor to the instructional designer. Content includes items such as learning objectives, instructor commentary or lectures, assignments, readings, and anything else that may be included in each lesson, as well as the course information (or syllabus).

The instructional designer and the instructor will decide on what each completion deadline includes after the initial design is complete and the workflow and format for getting content to the instructional designer.

Course Maintenance

ID begins the course maintenance process by contacting instructors, copying over courses into the spring course shells, and updating maintenance courses.

Resource: Instructional Design Wiki (Maintenance and Revision)

Video Production Complete

Following the media plan being set, a media specialist will partner with the instructor and instructional design to create any videos that are a part of that plan. A media specialist will have the instructor review the video planning tip sheet, fill out the media team video shoot requirements form, and provide an outline or script. Once these steps have been completed by the instructor then the media specialist will schedule a date and time for their video shoot.

Prior to the video shoot, the media specialist will determine what equipment is needed and how many media team members should attend the shoot based upon the information provided by the instructor. The media specialist will also call the instructor the week of their shoot prior to the actual day to touch base one final time and go over any last minute concerns or details. The day before the shoot the media specialist will prepare the equipment for the shoot. After video shoot has occurred, the media specialist will back up all footage to the media server prior to editing.

NOTE: Media will not schedule a video shoot until an instructor fills out the media team video shoot requirements and provides an outline or script.

Resources: Media Team Video Shoot Requirements and video planning tip sheet.
